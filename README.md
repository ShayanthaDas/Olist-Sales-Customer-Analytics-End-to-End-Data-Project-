# Olist Sales & Customer Analytics (End-to-End Project)

## Overview

This project analyzes real e-commerce data from Olist to understand sales performance, customer behavior, and delivery patterns. The goal was to connect different parts of the business (orders, customers, products, and sellers) and see how everything affects revenue and customer experience.

Instead of just doing basic analysis, I tried to build a complete end-to-end view of the business.

## What this project focuses on

* Overall sales performance trends
* Customer purchase behavior
* Delivery time and its impact on satisfaction
* Product category performance
* Seller contribution to revenue

## Tools used

* Python (Pandas, NumPy)
* SQL (data joining & querying)
* Power BI / Tableau (dashboarding)
* Matplotlib / Seaborn
* Jupyter Notebook

## Project workflow

1. Data understanding (multiple datasets)
2. Data cleaning and merging
3. SQL-based joins and exploration
4. Feature creation (delivery time, order value, etc.)
5. Customer and sales analysis
6. Dashboard development
7. Insights and recommendations

## Key insights

* Delivery time has a clear impact on customer satisfaction
* A small number of product categories generate most revenue
* Repeat customers contribute significantly to sales stability
* Some sellers consistently outperform others in both volume and rating

## What I learned

This was one of my more complete projects. It helped me understand how real e-commerce systems are structured and how different tables connect in real business data.

It also improved my skills in combining SQL + Python + dashboard tools in a single workflow.
